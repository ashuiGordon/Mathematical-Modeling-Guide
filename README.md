# 数学建模指南

> 一本面向数学建模竞赛的系统教程，涵盖理论基础、核心方法、高级技术与 AI Native 建模实战。

**在线阅读**：[https://ashuigordon.github.io/Mathematical-Modeling-Guide/](https://ashuigordon.github.io/Mathematical-Modeling-Guide/)

---

## 第一部分：基础知识

- **数学建模简介** — 什么是数学建模、建模基本流程、建模的意义与应用
- **数学基础** — 微积分、线性代数、概率统计、数值分析

## 第二部分：核心建模方法

- **评价模型** — 层次分析法（AHP）、灰色综合评价、模糊综合评价、BP 神经网络评价、数据包络分析（DEA）、主成分分析（PCA）、TOPSIS、组合评价法
- **优化模型** — 线性/整数/非线性/多目标规划、动态规划、微分方程组模型、图论与网络优化、概率优化模型、组合优化
- **分类模型** — 距离/Fisher/Bayes/逐步判别分析、系统聚类/K-means/模糊聚类/灰色聚类、SVM、决策树、随机森林、集成学习
- **预测模型** — 一元/多元/非线性/逻辑回归、ARIMA/季节分解/指数平滑/状态空间、灰色预测、BP 神经网络/SVR/深度学习/LSTM、组合预测

## 第三部分：高级建模技术

- **智能优化算法** — 遗传算法、模拟退火、粒子群优化、蚁群算法、差分进化、人工蜂群、混合智能算法
- **机器学习建模** — 监督学习、无监督学习、强化学习、深度学习、特征工程与选择、模型评估与选择

## 第四部分：AI Native 建模

- **AI 辅助建模工作流** — 端到端 7 阶段框架，覆盖从审题到论文的全流程
- **AI 辅助问题分析与建模思路** — 结构化 Prompt 审题、MECE 拆解、多角度分析
- **AI 辅助模型选择与设计** — 问题类型到模型族映射、多方案对比、模型创新
- **AI 代码生成与调试** — 自然语言/公式/伪代码转代码、AI 调试、IDE 集成
- **AI 辅助论文撰写** — 各部分写作 Prompt、LaTeX 排版、润色与翻译
- **Prompt Engineering 建模技巧** — 角色设定、思维链、自一致性、Prompt 链、建模专用模板库
- **Agent 编排与自动化建模** — 多 Agent 协作架构、RAG/MCP 技术、自动化 Pipeline
- **AI 建模工具生态** — 大模型对比、AI 编程助手、工具链方案、竞赛工具选型
